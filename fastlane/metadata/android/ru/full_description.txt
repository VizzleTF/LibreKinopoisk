MovieTorr — это Android-приложение для поиска торрентов по нескольким трекерам.
Приложение извлекает названия фильмов с веб-страниц и выполняет поиск по RuTracker, Kinozal, RuTor и NoNameClub.

Возможности:
- Поиск по нескольким трекерам
- Интерфейс Material Design 3 с поддержкой тем
- Встроенный браузер для навигации по сайтам
- Фильтрация по категориям
- Работа с magnet-ссылками

Технические особенности:
- Только FLOSS-зависимости
- Интеграция OWASP Dependency Check
- Обфускация ProGuard/R8
- Модульные тесты JUnit 5
- Безопасная работа с жизненным циклом фрагментов

Интеграция с API:
Приложение использует шлюз для запросов к торрент-трекерам, что позволяет:
- Получать данные с RuTracker, Kinozal, RuTor и NoNameClub без VPN
- Использовать единый API для всех трекеров
- Обеспечивать стабильную работу приложения

Приложение предназначено для образовательных целей и призывает использовать торрент-технологии только для легального контента.
